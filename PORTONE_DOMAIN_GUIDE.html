💳 패치 9 — PortOne 결제 + 도메인 연결 가이드

📌 이 패치는 실제 코드가 아닌 "준비 가이드" 입니다

PortOne 계정 정보와 도메인 정보가 있어야 실제 코드 적용 가능. 내일 PortOne 연결하실 때 이 가이드대로 진행하시면 됩니다.

PART 1: PortOne (포트원) 결제 연동

📋 사전 준비

1PortOne 계정 확인

  • ✅ portone.io 가입 완료
  • ✅ 사업자등록증 등록
  • ✅ PG사 계약 완료 (토스페이먼츠 추천)

2PortOne 대시보드에서 키 발급

  1. admin.portone.io 로그인
  2. 좌측 메뉴 결제 연동 → API 키
  3. 다음 키 복사:
    • Store ID (상점 ID)
    • Channel Key (채널 키)
    • API Secret (V2 시크릿)

3Vercel 환경변수 추가

Vercel Dashboard → chezsua → Settings → Environment Variables

NEXT_PUBLIC_PORTONE_STORE_ID=store-xxxxxxxx
NEXT_PUBLIC_PORTONE_CHANNEL_KEY=channel-key-xxxxxxxx
PORTONE_API_SECRET=secret-xxxxxxxxxxxxxxxxxxxxxxxxxxx

저장 후 Redeploy

📦 코드 추가 필요 (내일 작업)

실제 결제 작동을 위해 추가할 파일들:

파일역할
src/lib/portone.ts PortOne SDK 초기화
src/components/checkout/PaymentButton.tsx 결제 버튼 (모달 띄움)
src/app/api/payment/prepare/route.ts 결제 시작 - 주문 생성
src/app/api/payment/verify/route.ts 결제 검증 - 서버에서 확인
src/app/[locale]/checkout/page.tsx 결제 페이지
src/app/[locale]/checkout/success/page.tsx 결제 완료 페이지
src/app/[locale]/checkout/fail/page.tsx 결제 실패 페이지
💡 내일 진행 방법:
  1. 위 3가지 키를 알려주세요
  2. PG사 어느 곳 계약했는지 (토스/나이스/이니시스 등)
  3. 그러면 위 7개 파일을 만들어서 ZIP으로 보내드림
  4. 적용 후 테스트 결제 → 출시!

PART 2: 도메인 연결 (chezsua.com)

📋 사전 확인

1도메인 구매 확인

✅ chezsua.com 구매 완료 (사용자 확인)

도메인 등록 업체:

  • 가비아 / 카페24 / Namecheap / GoDaddy 등

🌐 Vercel에 도메인 연결

1Vercel에서 도메인 추가

  1. vercel.com/dashboard → chezsua 프로젝트
  2. Settings → Domains
  3. 입력란에 chezsua.com 입력 → Add
  4. 다시 www.chezsua.com도 추가

2DNS 설정

Vercel이 알려주는 DNS 정보를 도메인 등록 업체에 입력:

방법 1: A 레코드 (간단)

Type: A
Name: @
Value: 76.76.21.21

Type: CNAME
Name: www
Value: cname.vercel-dns.com

방법 2: Vercel 네임서버 사용 (자동, 추천)

도메인 등록 업체에서 네임서버를 다음으로 변경:

ns1.vercel-dns.com
ns2.vercel-dns.com
💡 한국 도메인 업체별:
  • 가비아: 도메인 관리 → DNS 정보 → 네임서버 변경
  • 카페24: 도메인 관리 → 네임서버 설정
  • Namecheap: Domain List → Manage → Nameservers

3DNS 전파 대기 (최대 24시간)

보통 1-2시간 안에 적용. 24시간 후에도 안 되면 DNS 설정 재확인.

확인 방법: 브라우저에서 https://chezsua.com 접속

4환경변수 업데이트

Vercel Dashboard → Settings → Environment Variables:

NEXT_PUBLIC_SITE_URL=https://chezsua.com

변경 후 Redeploy

5Supabase Auth URL 업데이트

Supabase Dashboard → Authentication → URL Configuration:

Site URL: https://chezsua.com
Redirect URLs:
- https://chezsua.com/**
- https://www.chezsua.com/**
- https://chezsua.vercel.app/** (백업)

🔍 SEO 등록 (출시 후)

Google Search Console

  1. search.google.com/search-console
  2. 속성 추가 → URL 입력 → chezsua.com
  3. 소유 확인 (HTML 파일 또는 DNS 인증)
  4. 사이트맵 제출: https://chezsua.com/sitemap.xml

네이버 웹마스터 도구

  1. searchadvisor.naver.com
  2. 네이버 로그인 → 사이트 등록
  3. chezsua.com 입력
  4. 소유권 확인
  5. 사이트맵 제출: https://chezsua.com/sitemap.xml

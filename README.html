🎯 룩북 Featured 관리하는 법

관리 페이지에서:

  1. /admin/lookbooks 접속
  2. 화면이 두 섹션으로 나뉘어 표시:
    • ⭐ Featured - 홈 고정된 룩북 (최대 5개)
    • 📋 모든 룩북 - 나머지 룩북
  3. 각 룩북 옆에 ⭐ 버튼 클릭 → 즉시 Featured 토글
  4. Featured 룩북은 순서 입력 칸이 표시됨
  5. 1, 2, 3 입력 → 홈 페이지에서 그 순서대로 표시
  6. 비워두면 자동 (최신순)

📦 자동 압축 작동 방식

  1. 4MB 이상 사진 선택 → 자동으로 압축됨
  2. 가로/세로 최대 2400px로 줄임
  3. 품질 92%부터 단계적으로 낮춤 (4MB 이하 될 때까지)
  4. 4MB 이하면 그대로 업로드

🧪 테스트

A. 큰 사진 업로드

  1. /admin/lookbooks/new 접속
  2. 10MB 이상의 사진 선택
  3. "압축 중..." → "업로드 중..." 진행률 표시
  4. 업로드 성공 → 썸네일 표시 ✅

B. 룩북 생성

  1. 모든 필드 입력 (Slug, Title EN, Client 필수)
  2. Create 버튼 → "저장되었습니다" ✅
  3. 목록으로 자동 이동

C. Featured 관리

  1. 룩북 목록에서 ⭐ 버튼 클릭 → Featured 토글
  2. Featured 섹션에 표시됨
  3. 순서 입력 (1, 2, 3...) → 홈 페이지 반영

🚨 문제 해결

여전히 업로드 안 되면:
  • Vercel Functions 로그 확인: vercel.com/dashboard → Functions
  • Supabase Storage → chezsua-images 버킷 권한 확인
  • F12 Console 에러 메시지 알려주기

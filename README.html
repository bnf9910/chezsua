  1. 홈 영상 크기 적당히 (max 800px)
  2. YouTube/MP4 URL 자동 임베드
  3. 중국어(ZH) 버튼 숨김

  7. 마이페이지 Admin 배너
  8. 샘플 데이터 정리 페이지 (/admin/cleanup)
  9. Insert Sample Data 버튼
  10. signout 페이지 (로그아웃 정상 작동)
  11. ★ AdminSidebar 수정 (대시보드 콘텐츠 보이게)

🔧 적용 방법 (5분)

STEP 1. SQL 실행 (선택)

이미 적용했으면 스킵 가능. 안 했으면:

  1. Supabase Dashboard → SQL Editor
  2. 01_video_column.sql 복사 → 붙여넣기 → Run

STEP 2. 파일 드래그 (가장 중요!)

윈도우 탐색기 두 개:

  • 왼쪽: 압축 푼 폴더
  • 오른쪽: C:\Users\BNF\Desktop\chezsua

왼쪽의 src와 messages 폴더를 → 오른쪽으로 드래그

"병합/덮어쓰기" 묻으면 예 / 덮어쓰기

STEP 3. 로컬 빌드 (선택)

cd C:\Users\BNF\Desktop\chezsua
npm run build

✓ Compiled successfully 나오면 OK!

STEP 4. Git Push

cd C:\Users\BNF\Desktop\chezsua

git add .

git commit -m "Apply all-in-one patch: dashboard fix + i18n + video + cleanup"

git push

STEP 5. Vercel 빌드 확인 (1-2분)

Vercel Dashboard → Deployments에서 🟢 Ready 확인

STEP 6. 시크릿 창에서 테스트

Ctrl + Shift + N → chezsua.vercel.app

  • 홈 → 영상 크기 적당히
  • 한국어 페이지 → 메뉴 영어
  • 로그인 → 메뉴 Dashboard → 대시보드 콘텐츠 보임 ★
  • /admin/cleanup → Insert Sample Data 클릭

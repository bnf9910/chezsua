🔧 적용 순서

1️⃣ SQL 실행

Supabase SQL Editor → 03_inquiry_and_users.sql 실행

  • project_inquiries 테이블 생성
  • users 테이블에 name, phone 등 추가

2️⃣ ZIP 압축 풀기 → src 드래그 (병합)

3️⃣ Vercel 환경변수 확인

Vercel Dashboard → chezsua → Settings → Environment Variables에 다음이 있어야 합니다:

RESEND_API_KEYResend에서 발급받은 키
INQUIRY_TO_EMAILchezsuaflower@gmail.com
INQUIRY_FROM_EMAILnoreply@chezsua.com 또는 onboarding@resend.dev

주의: Resend에서 chezsua.com 도메인 인증을 안 하셨다면, FROM_EMAIL을 onboarding@resend.dev로 두세요 (Resend 기본 도메인).

4️⃣ Git Push

cd C:\Users\(사용자명)\Desktop\chezsua
git add .
git commit -m "PHASE 1B: footer settings + terms/privacy + inquiry email"

🎯 테스트 방법

A. 푸터 정보

  1. 먼저 /admin/settings에서 전화번호, 이메일, 주소 등 입력 → Save
  2. 홈으로 가서 푸터 확인
  3. ✅ 입력한 정보가 푸터에 표시되면 성공!

B. 이용약관 / 개인정보 처리방침

  1. 푸터 하단의 이용약관 또는 개인정보 처리방침 클릭
  2. ✅ 실제 텍스트가 표시되면 성공!

C. Send Inquiry 이메일

  1. /project 페이지 접속
  2. 폼 작성 → Send Inquiry 클릭
  3. "Thank you. We will reply within 2 business days." 메시지 확인
  4. 🔍 chezsuaflower@gmail.com에서 메일 확인 (스팸함도!)
  5. ✅ 메일이 오면 성공!
💡 이메일이 안 오면?
  • 스팸함 확인
  • Vercel 환경변수 RESEND_API_KEY 확인
  • Resend Dashboard → API Keys에서 키 유효한지 확인
  • Resend Dashboard → Emails에서 로그 확인 (발송 시도 기록)

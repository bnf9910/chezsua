🚀 PHASE 1A — 룩북 상세 + SEO + Settings

매거진 클릭 시 큰 사진 + 갤러리, SEO 최적화, Settings 저장 기능

✅ 이 패치로 해결되는 것
  1. 🎯 매거진 클릭 시 전체 덮은 큰 사진 (히어로 이미지)
  2. 📸 사진 2장 이상이면 썸네일 갤러리
  3. 🔍 사진 클릭하면 라이트박스 확대 보기
  4. 📜 매거진 스타일 본문
  5. 🌍 SEO/GEO 최적화 (메타태그, Open Graph, JSON-LD)
  6. 🗺 sitemap.xml + robots.txt 자동 생성
  7. ⭐ 홈 페이지: Featured 룩북 먼저 + 최신순
  8. ⚙️ Settings 저장 기능 (실제 작동!)

🎯 사용 방법

1. 룩북 만들기 + 사진 업로드

  1. /admin/lookbooks/new 접속
  2. 📸 Images에 사진 여러 장 업로드
  3. 제목, 카테고리, 클라이언트 등 입력
  4. Featured 체크 → 홈에 표시됨
  5. Status: Published → Save

2. 매거진 클릭 시

홈 → 룩북 article → View Details 클릭

다음 화면이 보입니다:

  • ✨ 전체 화면 덮은 큰 사진 (85vh)
  • 📸 사진 여러 장이면 ← → 화살표 + 썸네일 갤러리
  • 🔍 사진 클릭하면 라이트박스 (✕로 닫기)
  • 📜 매거진 스타일 본문 (가운데 정렬, 큰 제목)
  • 📚 같은 카테고리 관련 룩북 3개

3. Settings 페이지에서 사이트 정보 변경

  1. /admin/settings 접속
  2. 사이트 정보, 연락처, 소셜, SEO 입력
  3. 하단 Save Settings 클릭
  4. "✓ 저장되었습니다" 메시지 확인

4. SEO 확인

구글 검색에서 사이트 찾을 수 있게 하려면:

  1. Vercel 배포 후 chezsua.vercel.app/sitemap.xml 접속 → XML 보이면 OK
  2. Google Search Console 사이트 등록
  3. 네이버 웹마스터 도구 사이트 등록
  4. sitemap.xml URL 제출
  5. 1-2주 후 검색 결과에 노출

🌍 SEO 작동 방식

요소역할
title구글 검색 결과 제목 + 브라우저 탭
description구글 검색 결과 설명
keywords검색 키워드 ("서울 플로리스트" 등)
Open Graph페이스북/카톡 공유 시 미리보기
JSON-LD구글 리치 결과 (별점, 이미지 등)
sitemap.xml모든 페이지 URL 목록 → 구글이 크롤링

💡 추가 SEO 팁

SEO 효과 극대화 방법
  • 매거진 글에 키워드 자연스럽게 넣기 (예: "강남 플로리스트", "PRADA")
  • 사진 alt 텍스트도 추후 SEO에 반영
  • 매거진 글이 최소 300단어 이상이면 좋음
  • 새로운 룩북 정기적으로 올리기 (월 2-3회)
  • 관련 룩북 링크가 자동으로 들어가서 좋음

🚨 주의사항

적용 후 확인할 것
  • 홈에 룩북 표시되는지 (없으면 "Stories Coming Soon")
  • 매거진 클릭 시 새 상세 페이지로 이동하는지
  • 사진 갤러리 작동하는지
  • Settings 저장 후 "✓ 저장되었습니다" 표시되는지
